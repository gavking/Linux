Hosts File

The hosts file is found in "/etc", it does not have an extension. When you request a URL, the system first looks here to see if it can get the IP address. By default it lists the loopback IP for localhost. You can setup redirects here for any domain name e.g. 127.0.0.1 google.com which redirect all calls to google.com to localhost. You can also create subdomains of localhost here e.g. 127.0.0.1 subdomain.localhost
